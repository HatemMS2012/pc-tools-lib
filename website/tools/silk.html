The Silk Link Discovery Framework is a tool for discovering relationships between data items within different Linked Data sources. It is available in different editions suiting different use cases.

Silk Single Machine can be used to produce link sets once and publish them along with the data. Silk Server can be set up to allow for linking to often changing link targets.

In order to use Silk Single Machine, you need:

  • The latest version of the Silk Link Discovery Framework.
  • The most recent Java Runtime Environment.

What to do:

  1. Write a Silk-LSL configuration file to specify which resources should be interlinked.
  2. Run Silk Single Machine: 
    java -DconfigFile= [-DlinkSpec=] [-Dthreads=] [-Dreload=(true/false)] -jar silk.jar
  3. Review Results: Open the output files designated in the Silk-LSL configuration and review the generated links.
