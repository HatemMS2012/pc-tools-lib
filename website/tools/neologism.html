Neologism is a web application that is based on the content manager Drupal. As such, it requires to have a working web server with some minimum requirements:

  • A Web server that can run PHP 5.2 or higher scripts and supports Drupal Clean URLs (the usage of mod_rewrite for Apache is recommended)
  • A MySQL server version 5.0.15 or higher
  • The PDO database extension for PHP

All-in-one packages such as WAMP and MAMP can be used to install all of the above on either a Windows or MacOS system.

Installation procedure:

  1. Uncompress the downloaded release of Neologism into a directory accessible from the Web server. For instance, if the Web server is configured to access content in the "srv" directory, unpack the content of the archive for Neologism into "srv/neologism".
  2. Create an empty database and name it with a properly name. For instance, "neologism-db".
  3. Open the address http://localhost/neologism in a web browser to start the installation procedure. In that address, replace "neologism" by the name of the directory you unpacked neologism into. The installer is a friendly procedure asking several questions about the configuration to generate. Screenshots of the differents steps are available on the Drupal web site.
